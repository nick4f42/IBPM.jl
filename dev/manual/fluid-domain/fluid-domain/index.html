IBPM.MultiGrid — Type
MultiGrid <: Grid

A type of Grid which uses a hierarchy of spacial discretizations to recursively solve until the solution converges. While faster than brute force solving large linear systems, MultiGrid typically uses relatively high memory allocation.

  • mg::Int: Number of domains in MultiGrid. Default: 1.
